# Noise Generator - Руководство по использованию

## Обзор

Noise Generator - это инструмент для генерации текстур на основе математических алгоритмов шумов. Инструмент доступен через меню **Tools > Noise Generator** в Unity Editor.

## Возможности

### Типы шумов

1. **Perlin** - Классический шум Перлина с FBM (Fractional Brownian Motion)
2. **Simplex** - Улучшенная версия шума Перлина
3. **Voronoi** - Диаграмма Вороного с различными режимами:
   - F1 - Расстояние до ближайшей точки
   - F2 - Расстояние до второй ближайшей точки
   - F1-F2 - Разность расстояний (создает края)
4. **Value** - Простой шум на основе значений
5. **Worley** - Вариант Voronoi шума
6. **White** - Белый шум (случайный)
7. **Billow** - Инвертированный абсолютный шум
8. **Ridged** - Шум с гребнями
9. **Cellular** - Клеточный шум

### Настройки изображения

- **Width/Height** - Размер текстуры (32-2048)
- **Format** - Формат текстуры (RGBA32, RGB24 и др.)
- **Generate MipMaps** - Генерация мипмапов
- Быстрые пресеты для квадратных текстур: 256, 512, 1024

### Параметры шума

- **Scale** - Масштаб шума (1-100)
- **Offset X/Y** - Смещение для генерации в разных областях
- **Use FBM** - Использовать фрактальное наложение
- **Octaves** - Количество октав (1-10)
- **Persistence** - Затухание амплитуды (0.1-1.0)
- **Lacunarity** - Увеличение частоты (1-4)
- **Seed** - Зерно для генерации (0 = случайное)

### Параметры Voronoi

- **Voronoi Type** - Тип вычисления
- **Cell Scale** - Масштаб ячеек

## Использование

1. Откройте **Tools > Noise Generator**
2. Настройте параметры изображения
3. Выберите тип шума
4. Настройте параметры шума
5. Нажмите **Generate Preview** для предварительного просмотра
6. Нажмите **Generate & Save** для сохранения текстуры

## Примеры применения

- Генерация ландшафтов
- Текстуры для материалов
- Процедурные атласы
- Визуальные эффекты
- Карты нормалей
- Ambient occlusion

## Технические детали

- Все шумы основаны на математических формулах
- Используется встроенный Perlin noise из Unity
- FBM позволяет создавать более детализированные текстуры
- Поддержка повторяющихся текстур через настройку wrap mode
